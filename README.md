## KANBAN PROJECT

Client Deploy : https://kanban-project-mct.netlify.app

Server Deploy : https://kanban-graphql-server.herokuapp.com/graphql

### Features

- Register, Sign In and Sign out
- Drag and Drop tasks
- Create, Update and Delete a task with conditions
- Create, Update and Delete a column with conditions

### Technologies and Packages

- Client

  - React
  - Hooks
  - Typescript
  - React Testing Library
  - Jest
  - React Router Dom
  - TailwindCSS
  - Apollo client
  - GraphQL
  - DND Beautiful with Test Utils
  - ANTDesign

- Server
  - Nodejs
  - Expressjs
  - ApolloServer
  - GraphQL
  - MongoDB
  - Mongoose
  - JsonWebToken
  - BodyParserGraphql
  - Bcrypt
  - ApolloServerExpress
  - ApolloServerErrors
